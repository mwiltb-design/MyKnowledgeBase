# About SQLite

SQLite is a C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine. SQLite is the most used database engine in the world. SQLite is built into all mobile phones and most computers and comes bundled inside countless other applications that people use every day. 

## Distinctive Features
- **Serverless**: Unlike most other SQL databases, SQLite does not have a separate server process. SQLite reads and writes directly to ordinary disk files. A complete SQL database with multiple tables, indices, triggers, and views, is contained in a single disk file.
- **Self-Contained**: A single library file contains the entire database system. No external dependencies are required.
- **Zero-Configuration**: No setup or administration is needed. 
- **Transactional**: All changes within a single transaction occur either completely or not at all, even if the program crashes or there is a power failure.
- **Open Source**: The SQLite source code is in the public domain.

## Usage and Reliability
SQLite is used as the on-disk file format for many desktop applications and is the standard database for Android and iOS. It is famous for its "Thoroughly Tested" status, with over 600 tests per line of code. The developers have committed to supporting the library through the year 2050.
