# Scope Expansion

## Summary
An exploration of the AutoResearch design pattern's versatility beyond machine learning, applying it to domains like trading, marketing, and web performance.

- Definition: **[[AutoResearch]]** is a design pattern for **any measurable task**.
- Non-ML Use Cases:
    - **Marketing**:
        - Ad creatives, landing pages, and email copy.
        - Metric: **[[Metric Definition|Conversion Rate]]**.
    - **Trading**:
        - Buy/sell rules and market strategy.
        - Metric: **[[Metric Definition|Sharp Ratio]]**.
    - **Web Performance**:
        - Optimize loading times and efficiency.
        - Metric: **[[Metric Definition|Loading Time (ms)]]**.
    - **Prompt Engineering**:
        - Fine-tune system instructions for **[[Pi Coding Agent|AI Agents]]**.
- **The Singularity**: Recursive self-improvement loops that run distributed across networks.
