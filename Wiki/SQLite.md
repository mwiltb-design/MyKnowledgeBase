## Summary
- **SQLite** is a lightweight, serverless SQL database engine used for agent state and memory persistence.
- **FTS5 Extension**: Utilized by **[[Hermes Agent]]** for high-speed full-text search over conversation histories.
- Provides a reliable, local-first storage layer for **[[AI Memory]]**.
