# OxyLabs MCP

## Summary
A look at the OxyLabs Model Context Protocol (MCP) integration, which provides AI agents with robust web-scraping and data automation capabilities.

- Definition: **OxyLabs MCP** is an integration for **[[AutoResearch|AI agents]]** to gain web-scraping powers.
- Key Capabilities:
    - **Live Web Access**: Pulls structured data from Amazon, Google, and real estate listings.
    - **Automation**: Compatible with **n8n** for no-code workflows.
    - **Integration**: Connects directly to **[[Pi Coding Agent|CLI agents]]** like Codex or Claude Code.
- Functionality: Handles **proxy rotation, captcha solving, and JavaScript rendering**.
- Use Case: Scraping competitor pricing or real-time stock data for analysis.
