## Summary
- Open-source protocol for **declarative, secure, and interactive** user interfaces generated by AI agents.
- Decouples AI "brains" from native UI "bodies" using a **Component Catalog** system.
- Employs a flat **Adjacency List Model** for incremental, LLM-friendly streaming.

## Architecture & Data Flow
- **Interaction Loop:** User Input -> Agent Generation (JSON) -> Client Streaming -> Native Rendering (Native Widgets).
- **Adjacency List:** Represents UI as a flat list of components to prevent "JSON hallucination" and simplify incremental updates.
- **Transports:** Supports **[[OxyLabs MCP]]**, WebSockets, and HTTP streams.

## Protocol (v0.8 Stable)
- **JSON Format:** Flat, streaming-capable structure avoiding deep nesting.
- **Data Binding:** Supports mapping UI elements to dynamic states without full re-transfers.

## Security
- **No Code Execution:** Agents send data (JSON intents), never logic (JS/Python).
- **Pre-approved Components:** Client only renders from a local **Component Catalog**.
- **Sandbox:** Inherits host application permissions and security boundaries.
